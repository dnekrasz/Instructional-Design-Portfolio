If we wish to find the value of a negative z-score, we can take advantage of the fact that the standard normal distribution is symmetrical about 0 to say that \(Φ(-z) = 1 - Φ(z)\). In the example above, \(Φ(-0.75) = 1 - 0.773 = 0.227\).

What’s more, we can use z-scores to determine how likely \(X\) is to lie between to values. If, for example, we want to know how likely \(X\) is to be between 25 and 40 (as in Figure 4), and \(X\sim N(30, 25)\), we can say

$$ Pr(25 ≤ X ≤ 40) = F_X(40) - F_X(25) $$

$$ = Φ\left(\dfrac{40-30}{5}\right) - Φ\left(\dfrac{25-30}{5}\right) = Φ(2) - Φ(-1) = 0.977 - (1 - 0.841) = 0.818 $$
